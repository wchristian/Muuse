=pod

=head1 NAME

Muuse - Moose but with less typing

=head1 VERSION

version 1.191300

=head1 SYNOPSIS

    use Muuse;
    
    ro "hro";
    lazy hlazy => sub { 2 };
    rwp "hrwp";
    rw "hrw";

=head1 DESCRIPTION

Muuse imports both L<Moose> and L<MooseX::ShortHas>, making it even less work in
typing and reading to set up an object.
